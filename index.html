About

I work with large-scale data pipelines, transforming raw data into meaningful insights using the Azure ecosystem. My role involves integrating data from multiple sources—SharePoint,SAP, Salesforce and SFTP feeds—into Azure Data Lake Storage Gen2 (ADLS Gen2). Using Azure Data Factory (ADF), I orchestrate data movement, and in Azure Databricks, I implement the medallion architecture (Bronze → Silver → Gold) to clean, transform, and optimize datasets. The refined data is stored in Delta Tables or used to populate analytical tables. On top of this, I design efficient views that power interactive Power BI reports, enabling data-driven decision-making. Beyond development, I handle real-time issue resolution via ServiceNow and Azure DevOps Boards, ensuring seamless data workflows. My expertise spans ETL automation, cloud data engineering, performance tuning, and analytics, driving scalable and efficient data solutions.
